# GlobPack (gp) 📦
**GlobPack** is a lightweight, community-driven global package manager for Termux and other Linux-based terminals.

## How it Works
GlobPack uses a decentralized registry system. Anyone can contribute by forking this repo and adding their package to the `registry/` folder.

## Installation

Install GlobPack globally using npm:

```bash
npm install -g globpack
